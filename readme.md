# Telegram group data 

## 1. Get telegram group name from "result.json" file

Structure of "result.json" file:
```json
{
    "name": "Group name",
    "type": "group",
    "id": "group id",
    "messages": [
        {
            "id": "message id",
            "from": "user name",
            "text": "message text",
            "date": "message date"
        },
        {
            "id": "message id",
            "from": "user name",
            "text": "message text",
            "date": "message date"
        }
    ]
}
    
```
